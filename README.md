# Password Generator

A modern Password Generator application built using Java Swing and JFrame.  
This desktop application generates strong and secure passwords through a clean graphical user interface.

## Features

- Generate strong random passwords
- Simple and modern GUI
- Built using Java Swing and JFrame
- Lightweight desktop application
- Beginner-friendly project structure
- Fast password generation

## Technologies Used

- Java
- Swing
- JFrame
- OOP Concepts

## How to Run

1. Clone the repository

```bash
git clone https://github.com/Pradeeeeeeeep/PasswordGenerator.git
```

2. Open the project in IntelliJ IDEA or VS Code

3. Compile the source files

```bash
javac src/*.java
```

4. Run the application

```bash
java src/App
```
